# Codex Skills

Personal Codex skills for AI + medicine literature learning and terminology workflows.

## Skills

- `paper-to-terms`: Convert papers or Zotero collections into Obsidian literature notes and linked term notes.
- `term-vault`: Maintain a lightweight Obsidian terminology library with English canonical terms and Chinese explanations.
- `term-deep-dive`: Expand existing term notes into linked long-form study notes.
